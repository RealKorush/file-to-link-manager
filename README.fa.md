# 📂 ربات تلگرام مدیریت فایل‌های S3 Compatible (لیارا)

این یک ربات تلگرامی پیشرفته و پایدار است که با زبان پایتون توسعه یافته و به ادمین‌های سرور اجازه می‌دهد تا فایل‌های موجود در باکت‌های ذخیره‌سازی ابری S3 (مانند سرویس ذخیره‌سازی ابری لیارا) را مستقیماً از داخل تلگرام مدیریت کنند.

## ✨ قابلیت‌ها
- 🔐 **امنیت بالا:** قفل اختصاصی ادمین ارشد (فقط افراد مجاز توانایی تعامل دارند).
- 📊 **مشاهده ظرفیت:** نمایش حجم مصرف شده، تعداد کل فایل‌ها و فضای باقی‌مانده به تفکیک هر باکت به صورت لحظه‌ای.
- 🔗 **لینک دانلود موقت:** تولید سریع لینک‌های دانلود امن دارای انقضا (Presigned URLs).
- ✏️ **تغییر نام آسان:** تغییر نام فایل‌های داخل باکت بدون نیاز به دانلود و آپلود مجدد.
- 🗑️ **حذف ایمن:** حذف فایل‌ها از باکت ابری همراه با تاییدیه دو مرحله‌ای جهت جلوگیری از خطای انسانی.
- ⚙️ **ماشین وضعیت پایدار:** استفاده از `ConversationHandler` برای پایداری ۱۰۰٪ ربات در مقابل کلیک‌های ناگهانی.
